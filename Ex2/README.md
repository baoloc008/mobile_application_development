# Currency Converter App

## Technology

* Android
* Retrofit (2.4.0)
* Converter-gson (2.4.0)
* Api: [https://openexchangerates.org/](https://openexchangerates.org/)

## Usage

* Click `REFRESH` to get rate from api again.
* Click `CLEAR` to reset fields.
* Click `CONVERT` to convert money.
